# AI-Powered Stock Market Analysis & Paper Trading Simulator

An intelligent stock market analysis and virtual trading platform that combines technical analysis with AI-driven insights to help users understand market trends and practice trading without financial risk.

## Features

* 📈 Real-time and historical stock market data analysis
* 🤖 AI-powered market sentiment analysis using Google Gemini API
* 📊 Technical indicators including RSI, MACD, Moving Averages, and Bollinger Bands
* 💡 Trend-based stock recommendations and market insights
* 💰 Paper trading simulator for virtual stock buying and selling
* 📂 Portfolio management with profit/loss tracking
* 🗄️ MySQL database integration for stock data, portfolios, transactions, and prediction history
* 📉 Performance analytics and trading history visualization

## Tech Stack

* Java
* MySQL
* Google Gemini API
* Technical Analysis Algorithms
* JDBC

## Project Objective

This project aims to provide a risk-free environment where users can learn stock market trading, analyze market trends using technical indicators, and leverage AI-generated insights to make informed investment decisions. The paper trading feature allows users to test trading strategies without using real money.

## Key Functionalities

1. Historical stock data processing and analysis
2. AI-assisted market sentiment evaluation
3. Technical indicator calculation and visualization
4. Virtual portfolio creation and management
5. Buy/Sell stock simulation
6. Profit and loss calculation
7. Transaction and prediction record management

## Future Enhancements

* Real-time stock market data integration
* Advanced charting and visualization
* Machine learning-based price prediction
* Multi-user authentication and dashboards
* Risk management and portfolio optimization tools
